# Real Estate Market Trends Dashboard

## CodeAlpha Power BI Internship

### Project Overview

This dashboard analyzes real estate market trends, property pricing, rental yields, market demand, and property supply using Power BI.

### Features

* Average Property Price Analysis
* Property Supply by Property Type
* Rental Yield Analysis
* Market Demand Trend
* City and Property Type Filters
* Interactive Dashboard

### Tools Used

* Power BI Desktop
* Microsoft Excel
